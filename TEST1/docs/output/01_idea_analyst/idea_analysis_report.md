# 아이디어 분석 보고서

## 1. 프로젝트 개요

- **프로젝트명:** AgileFlow — AI 에이전트 소프트웨어 팀 플랫폼
- **한줄 설명:** 자연어 요구사항 하나로 기획·설계·개발·테스트·배포까지 자율 수행하는 멀티 에이전트 소프트웨어 팀
- **핵심 가치:** 개인 개발자 또는 소규모 팀이 전문 인력 없이 풀스택 소프트웨어 프로젝트를 완주할 수 있도록 AI 에이전트 파이프라인이 전 공정을 대행

---

## 2. 타겟 사용자

| 페르소나 | 연령대 | 니즈 | 페인포인트 |
|---------|--------|------|-----------|
| 1인 스타트업 창업자 | 25~40대 | 아이디어를 빠르게 MVP로 구현 | 개발자 고용 비용, 느린 초기 개발 속도 |
| 비개발자 기획자 | 28~45대 | 기획서를 바로 작동하는 코드로 변환 | 개발팀과의 커뮤니케이션 장벽 |
| AI/ML 연구자 | 25~35대 | LangGraph·MCP 멀티 에이전트 학습 실습 | 복잡한 에이전트 설계를 쉽게 실험할 환경 부재 |

## 3. 핵심 기능 목록

### MVP (v1)
- [x] 자연어 아이디어 입력 → 분석 보고서 자동 생성 (01_idea_analyst)
- [x] 기획서(PRD, User Story, API 명세) 자동 작성 (02_planner)
- [x] 디자인 시스템 + 와이어프레임 자동 생성 (03_designer)
- [x] 기술 스택 선정 + 시스템 아키텍처 설계 (04_architect)
- [x] 프론트엔드 컴포넌트·페이지 코드 생성 (05_frontend_dev)
- [x] 백엔드 API·서비스 코드 생성 (06_backend_dev)
- [x] 단위/통합/E2E 테스트 코드 자동 생성 (07_tester)
- [x] Docker·CI/CD·인프라 구성 자동화 (08_deployer)
- [x] 전체 파이프라인 조율 + 최종 리포트 (00_orchestrator + 09_project_manager)

### 확장 기능 (v2+)
- [ ] Human-in-the-loop: 중간 단계 사람 승인 인터페이스
- [ ] Time Travel: 과거 상태로 롤백하여 다른 방향 재시도
- [ ] MCP 커스텀 서버 연동 (외부 API·DB 실시간 연결)
- [ ] LangSmith 기반 에이전트 트레이싱 대시보드
- [ ] 다중 프로젝트 병렬 실행 및 팀 공유

## 4. 경쟁 분석

| 서비스명 | 장점 | 단점 | 차별점 |
|---------|------|------|--------|
| Devin (Cognition AI) | 자율 코딩 에이전트 선두 | 블랙박스, 커스터마이징 불가 | 오픈 파이프라인, 단계별 투명성 |
| GitHub Copilot | IDE 내 코드 자동완성 | 단일 파일 수준, 전체 설계 불가 | 전체 SDLC(기획~배포) 커버 |
| LangChain Templates | 에이전트 빌딩 블록 | UX 없음, 파이프라인 직접 조립 필요 | 완성된 워크플로우 + UI 제공 |

## 5. 기술 리스크 & 제약사항

- **LLM 비결정성:** 동일 입력에도 매번 다른 코드 생성 가능 → 출력 검증 레이어 필수
- **컨텍스트 길이 한계:** 대형 프로젝트에서 이전 단계 산출물이 컨텍스트를 초과 → 요약 압축 전략 필요
- **코드 실행 안전성:** 생성된 코드를 자동 실행 시 보안 위험 → Docker Sandbox 격리 필수
- **비용:** 9단계 파이프라인 1회 실행 시 고비용 API 호출 → 캐싱·모델 라우팅 최적화 필요

## 6. 프로젝트 규모 판단

- **예상 규모:** 중
- **난이도:** 상 (LangGraph 상태 관리 + MCP 통합 + 에이전트 간 통신 설계)
- **권장 기술 스택 방향:**
  - 오케스트레이션: LangGraph (Python)
  - 에이전트 연결: MCP (Model Context Protocol)
  - 트레이싱: LangSmith
  - 프론트엔드: Vite + React (에이전트 실행 모니터링 UI)
  - 백엔드: Fastify (파이프라인 API 서버)
  - 실행 격리: Docker Sandbox
